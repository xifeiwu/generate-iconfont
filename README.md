### 简介
将若干svg文件打包成eot, woff格式的字体。

**在前端，使用字体与使用svg或其它格式的图片对比**

优点：

- 可以设置大小，颜色
- 一次传输，减少网络请求

缺点：

- 字体颜色只能是一种颜色，不支持两种及以上的颜色


**svg打包方案**

按需求：单色svg打包成字体；多个颜色通过svg symbol的方式实现
